=============================================
comandos de manipulação de diretórios 

Todos comandos em linux é minusculo 

Comando ls: é listar arquivos ou diretórios 
 o sinal de  "- ou seja menos" segnifica passar uma opção 

 ls -a: segnifica mostrar tudo 
 ls -lha: ver os arquivos detalhadamente de forma humana 

 Todo arquivo iniciar com ponto, ele está oculto

 Branco: arquivo de texto
 Azul:

 cd: mudar de diretórios
 pwd: qual diretório você está

 Cor Azul calcinha: link simbolico, você está vendo mais não existe
 Cor marca verde texto: Perigo 

 Simbolo "/" segnifica que você estpa na RAIZ

 comando (cd -) só volta apenas uma vez
 cd~: o tio, representa sempre seu diretório pessoal 
 cd ..: Segnifica descer um nivel 
 pwd: qual diretório você está
 History: Mostra o histórico de comandos utilizados 
!: segnifica executar o comando da linha de referência 
mkdir: comando para criar diretórios
  mkdir -v: segnifica verboso , ele mostra oque você está fazendo (detalhado) e com algum nome na frente cria pasta

  Criar um diretório SENAC Tatuapé

  mkdir -v + Aspas simples no inicio e no final ' ', cria uma pasta com duas palavras ou seja, diretório

  ajuda de comandos 

  --help

  para criar uma pasta SENAC e dentro da pasta senac criar outras: mkdir -pv tat/info

  rm - remover arquivos ou diretórios 
  -f : segnifica forçar
  -v: segnifica verboso
  -R: segnifica recursivo
  rm -Rfv Alisson/: exclui a pasta
